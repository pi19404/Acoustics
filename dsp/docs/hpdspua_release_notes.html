Introduction   Documentation    What’s New   Upgrade and Compatibility Information   Host Support and Validation Information Device Support  


Introduction

This is the High Performance DSP Utility Application (HUA). It demonstrates the use of various Texas Instruments Software Development Kits including BIOS 6, the Network development Kit (NDK), the Chip Support Library (CSL) and the Platform Library. 

The utility consists of one Package :

  • hua: This is the utility program.

Everything is provided as source and there are pre-built executables (debug and release) that can be used without re-building.



Documentation

Documentation consists of these Release Notes which contain the most up to date information and the Users Guide.



What’s New

This is a  release of this application for the TMDXEVM6678L and TMDXEVM6670L Platforms.

A network loopback was added starting 2.0.5. See the benchmarks tab.

Host Support and Validation Information

This release was built using Code Composer Studio on Microsoft Windows XP SP3 and uses the following components (shown with minimum revision level):

Component Version
Code Composer Studio (CCS) 5.0.3
Codegen Compiler (part of CCS) 7.2
SYS/BIOS 6.31.04.27, 6.32.01.38
Network Developers Kit (NDK) 2.20.4.26, 2.20.04.26
XDC 3.20.03.63, 3.22.01.21
TMDXEVM6678L Platform Development Kit (PDK) 1.0
Multi-core Software Development Kit (MCSDK) 2.0
   

Warning: If you use other versions of these components, you must make sure that ALL of them are compatible with each other or it will lead to build errors. If you have upgraded these components through a release of MCSDK you should be okay.



Device Support
